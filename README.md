# sql-data-warehouse
A data mining project that builds a data warehouse from multiple datasets using SQL Server for data cleaning, transformation, and integration, followed by preparing a final dataset for machine learning applications.
This project focuses on building a Data Warehouse using multiple healthcare datasets, including Diabetes, Heart Disease, and Hypertension Risk Prediction data.
Data preprocessing, cleaning, and integration are performed using SQL Server to produce a unified and structured dataset.
The final dataset is then prepared for Machine Learning tasks to support disease risk prediction and data-driven analysis.
